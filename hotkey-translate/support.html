HotKey Translate Support

Last updated: May 6, 2026

HotKey Translate is a macOS menu bar app for quick text translation. This page provides basic support information for users.

Getting Started

Copy text to the macOS clipboard, then press the configured global hotkey. By default, the shortcut is Shift + Command + X. The translator opens from the menu bar.

Changing the Hotkey

Right-click the HotKey Translate menu bar icon, choose Settings, click the hotkey field, and press a new shortcut.

Launch at Login

You can enable or disable Launch at Login from the Settings window.

Troubleshooting

  • Make sure the text you want to translate is copied to the clipboard.
  • If the hotkey does not work, choose a different shortcut in Settings.
  • If translation does not load, check your internet connection and try again.
  • If Google Translate changes its web interface, the embedded translation view may require an app update.
